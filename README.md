# Etch-a-Sketch

Sketch drawing on the canvas by hovering over tiles.

![Sketch](/screenshots/sketch_1.png)

 The canvas is 16x16 by default. Canvas can be resized with the "Resize" button, inputting number of squares per side, between 1-100.

![Resize](/screenshots/resize.png)

![Resize Prompt](/screenshots/resize_prompt.png)

![Resized Grid](/screenshots/resized_grid.png)

Canvas can be cleared with the "Reset" button.

![Clear](/screenshots/clear.png)

![Cleared Grid](/screenshots/cleared_grid.png)
